## Targets

- **Grammar spine:** the comparison structure **A + 比 + B + Adjective**. 他比我高 = *He is taller than me*.
- **Negation:** for comparisons, use **A + 没有 + B + Adjective**. 我没有他高 = *I'm not as tall as him*.
- Common slip: never insert 很 inside a 比 sentence. ✘ 他比我**很**高 → ✓ 他比我**更**高 (*even taller*).
- **Vocab cluster:** comparative-adjective pack — 快 (fast), 慢 (slow), 贵 (expensive), 便宜 (cheap), 长 (long), 短 (short).
