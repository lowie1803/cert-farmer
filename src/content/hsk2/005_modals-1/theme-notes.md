# Week 5 — Modals I (想 / 要 / 会)

> 🚧 **Authoring in progress.**

## Targets

- **想 (xiǎng)** — *want to / would like to*. Polite, soft. 我想喝水.
- **要 (yào)** — stronger *want / will / need to*. 我要回家.
- **会 (huì)** — *know how to* (learned skill) **and** *will* (likelihood / prediction). 我会说中文. 明天会下雨.
- **Negation:** 不想, 不要, 不会 (all with 不, not 没).
- **Vocab cluster:** action verbs that pair naturally with modals — 开车 (drive), 游泳 (swim), 跳舞 (dance), 唱歌 (sing).
