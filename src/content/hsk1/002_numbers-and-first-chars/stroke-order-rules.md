# Stroke-Order Rules

Stroke order matters less for reading and more for **muscle memory**, **handwriting legibility**, and — surprisingly — for typing on handwriting-input keyboards. If you ever plan to write Chinese by hand, learn the rules now; if not, at least learn to *recognize* the canonical order so character lookup apps work.

The seven core rules cover ~95% of cases:

| # | Rule | Example |
|---|------|---------|
| 1 | Top → bottom | 三: top — middle — bottom |
| 2 | Left → right | 川: left — middle — right vertical |
| 3 | Horizontal before vertical (when they cross) | 十: 一 then 丨 |
| 4 | Outside before inside | 月: left vertical, top-right hook, then the two insides |
| 5 | Close the box last | 国: 冂 first, contents 玉, then bottom closes |
| 6 | Center before symmetric flanks | 小: center stroke, then left & right dots |
| 7 | The slash-down to the left comes before the slash-down to the right | 人: 丿 first, then 乀 |

## The eight basic strokes

| Stroke | Name | Looks like |
|--------|------|------------|
| 横 héng | horizontal | ─ |
| 竖 shù | vertical | │ |
| 撇 piě | left-falling | ╱ |
| 捺 nà | right-falling | ╲ |
| 点 diǎn | dot | ﹒ |
| 提 tí | rising | ⤴ |
| 折 zhé | turning (combo) | corners and hooks |
| 钩 gōu | hook | ╮ |

Every more complex stroke is a combination of these eight. The character 永 (*yǒng*, "eternal") famously contains all eight — calligraphy students drill it for that reason.

## Why this *actually* matters for you

You will not become a calligrapher this week. The reason to know the rules is:

- **Memory.** Writing a character with the right order locks it into motor memory; copying it in the wrong order does not.
- **Search.** Handwriting input on phones and dictionaries (Pleco, Google IME) is much better when your order is canonical — they ignore minor sloppiness, not wrong order.
- **Components.** Most stroke-order rules are component rules in disguise (e.g. "outside before inside" is just "finish the enclosing component before its contents"). Knowing them makes character breakdowns feel obvious instead of mysterious.

**This week's drill (weekend, optional):** write 一 二 三 四 五 六 七 八 九 十 in canonical order, ten times each. If you write digits 0–9 from muscle memory in English, this is the equivalent.
