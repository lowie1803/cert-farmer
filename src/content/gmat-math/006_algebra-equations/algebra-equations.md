# Algebra & Linear Equations

## The Golden Rule of Equations

Whatever you do to one side, do to the other.

Goal: isolate the variable.

## Solving Linear Equations

**One variable**:

Solve 3x − 7 = 14:
1. Add 7 to both sides: 3x = 21
2. Divide by 3: x = **7**

**With fractions** — multiply through by the LCD first:

Solve x/3 + x/6 = 5:
- LCD = 6. Multiply: 2x + x = 30 → 3x = 30 → x = **10**

**With distribution**:

Solve 3(x − 2) = 2(x + 4):
- 3x − 6 = 2x + 8
- x = **14**

## Systems of Two Equations

Two equations, two unknowns. Methods: substitution or elimination.

### Substitution

Solve for one variable, substitute into the other:

2x + y = 10  
x − y = 2

From equation 2: x = y + 2. Substitute into equation 1:
2(y+2) + y = 10 → 2y+4+y = 10 → 3y = 6 → y = 2, x = 4.

### Elimination (Addition/Subtraction)

Add or subtract equations to cancel a variable:

2x + y = 10  
x − y = 2

Add the two equations: 3x = 12 → x = 4 → y = 2.

**Tip**: If coefficients don't cancel directly, multiply one equation by a constant first.

## Special Cases

- **No solution**: Parallel lines (same slope, different intercepts). E.g., 2x+y=5 and 4x+2y=8 → 4x+2y=10≠8.
- **Infinite solutions**: Same line. E.g., 2x+y=5 and 4x+2y=10.

## Translating Word Problems into Equations

| English | Math |
|---------|------|
| "is", "equals", "is the same as" | = |
| "more than", "increased by" | + |
| "less than", "decreased by" | − |
| "of", "times", "product of" | × |
| "per", "for each" | ÷ |

> Two numbers sum to 45 and differ by 9. Find the larger.

x + y = 45 and x − y = 9. Add: 2x = 54 → x = **27**.

## Worked Examples

### Example 1: Variable on both sides
Solve (x−3)/2 = (x+1)/4:
- Multiply through by 4: 2(x−3) = x+1
- 2x − 6 = x + 1
- x = **7**

### Example 2: System by elimination
3x + 2y = 16  
x − 2y = 0

Add: 4x = 16 → x = 4 → y = 2.

### Example 3: Two-variable word problem
Adult tickets cost $8, children's $5. 50 tickets sold for $310 total. How many adult tickets?

a + c = 50 and 8a + 5c = 310. From the first: c = 50−a. Substitute:
8a + 5(50−a) = 310 → 3a = 60 → a = **20**.

## Common Traps

- **Distribution**: 3(x−2) = 3x−6, not 3x−2.
- **Sign errors**: When subtracting an equation, flip all signs in that equation.
- **Fractional equations**: (x+3)/2 ≠ x/2 + 3. It equals x/2 + 3/2.
- **"Less than" direction**: "5 less than x" = x−5, not 5−x.
