# Inequalities & Absolute Value

## Solving Inequalities

Inequalities work like equations **except**: multiplying or dividing by a **negative number flips the sign**.

Solve −3x > 12:
- Divide by −3 (flip!): x < −4

Solve 5 − 2x ≥ 11:
- −2x ≥ 6
- Divide by −2 (flip!): x ≤ **−3**

## Compound Inequalities

Work on all three parts simultaneously:

Solve −2 < 3x − 1 ≤ 8:
1. Add 1 to all parts: −1 < 3x ≤ 9
2. Divide by 3: **−1/3 < x ≤ 3**

## Quadratic Inequalities

x² − 4 > 0 → (x−2)(x+2) > 0

Roots are x = 2 and x = −2. Test regions:
- x < −2: (−)(−) = positive ✓
- −2 < x < 2: (−)(+) = negative ✗
- x > 2: (+)(+) = positive ✓

Answer: **x < −2 or x > 2**

## Absolute Value

|x| = distance from 0 on the number line.

|x| = 5 means x = 5 or x = −5.

### Solving |expression| = k

Split into two cases:
|3x − 6| = 12:
- Case 1: 3x − 6 = 12 → x = **6**
- Case 2: 3x − 6 = −12 → 3x = −6 → x = **−2**

### Solving |expression| < k (between)

|x − 3| < 5 means −5 < x−3 < 5:
- Add 3: **−2 < x < 8**

### Solving |expression| > k (outside)

|x − 3| > 5 means x−3 > 5 OR x−3 < −5:
- **x > 8 or x < −2**

**Memory trick**: Less than → "and" (between). Greater than → "or" (outside).

## Key Inequality Properties

| Situation | Rule |
|-----------|------|
| Add/subtract same value | Inequality direction unchanged |
| Multiply/divide by positive | Unchanged |
| Multiply/divide by negative | **Flip the sign** |
| Square both sides | Only valid if both sides non-negative |

> If a < b, does −a < −b? No! −a > −b. Negating flips the inequality.

## What "Must Be True" Problems

With inequalities, test extreme values:

> If a < b, which must be true?

- a² < b²? No — if a=−5, b=3: 25 > 9.
- |a| < |b|? No — same example.
- a − 1 < b − 1? **Yes** — subtracting 1 from both sides preserves the inequality.

## Worked Examples

### Example 1
Solve: 2x − 3 ≤ 7 and x > 0
- From first: x ≤ 5. Combined with x > 0: **0 < x ≤ 5**

### Example 2
|2x + 1| = 7:
- 2x + 1 = 7 → x = 3
- 2x + 1 = −7 → x = −4
- Solutions: **x = 3 and x = −4**

### Example 3
For which x is x² − 4 > 0?
- (x−2)(x+2) > 0 → **x < −2 or x > 2**

## Common Traps

- **Forgetting to flip**: −x > 5 → x < −5, not x > −5.
- **Squaring both sides**: Only valid when both sides are non-negative.
- **|a + b| vs |a| + |b|**: Triangle inequality — |a+b| ≤ |a| + |b|.
- **Division by variable**: Never divide by a variable in an inequality unless you know its sign.
