# Exponents & Roots

## Exponent Rules

| Rule | Formula | Example |
|------|---------|---------|
| Product | xᵃ × xᵇ = xᵃ⁺ᵇ | 2³ × 2⁴ = 2⁷ = 128 |
| Quotient | xᵃ / xᵇ = xᵃ⁻ᵇ | 3⁵ / 3² = 3³ = 27 |
| Power of a power | (xᵃ)ᵇ = xᵃᵇ | (2³)⁴ = 2¹² |
| Power of a product | (xy)ᵃ = xᵃyᵃ | (2×3)² = 4×9 = 36 |
| Zero exponent | x⁰ = 1 (x≠0) | 7⁰ = 1 |
| Negative exponent | x⁻ⁿ = 1/xⁿ | 2⁻³ = 1/8 |

## Fractional Exponents

$$x^{m/n} = \left(\sqrt[n]{x}\right)^m = \sqrt[n]{x^m}$$

- 8^(1/3) = ∛8 = 2 (cube root)
- 16^(3/4) = (16^(1/4))³ = 2³ = **8**
- 27^(2/3) = (∛27)² = 3² = **9**

**Strategy**: First take the root (reduces the number), then raise to the power.

## Simplifying Radicals

To simplify √n: factor out perfect squares.

$$\sqrt{48} = \sqrt{16 \times 3} = 4\sqrt{3}$$

$$\sqrt{75} = \sqrt{25 \times 3} = 5\sqrt{3}$$

### Multiplying Radicals

$$\sqrt{a} \times \sqrt{b} = \sqrt{a \times b}$$

√12 × √3 = √36 = 6

### Adding/Subtracting Radicals

Only combine **like radicals** (same radicand):

3√5 + 2√5 = 5√5 ✓

3√5 + 2√3 = cannot simplify ✗

### Rationalizing the Denominator

To remove √ from the denominator, multiply by √/√:

$$\frac{6}{\sqrt{3}} = \frac{6}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} = \frac{6\sqrt{3}}{3} = 2\sqrt{3}$$

## Working with Variables

$$(x^3 \times x^{-1}) \div x^2 = x^{3+(-1)-2} = x^0 = 1$$

When multiplying/dividing same base: add/subtract exponents. Never confuse (xᵃ)(xᵃ) = x²ᵃ with (xᵃ)² = x²ᵃ — they're the same! But (xᵃ)(yᵃ) = (xy)ᵃ, not xᵃ + yᵃ.

## Worked Examples

### Example 1: Simplify (2³ × 2⁻¹) / 2²
= 2^(3-1-2) = 2⁰ = **1**

### Example 2: What is 64^(2/3)?
= (64^(1/3))² = 4² = **16**

### Example 3: Simplify √(12) × √(27)
= √(12 × 27) = √324 = **18**

Or: √12 = 2√3, √27 = 3√3, product = 6×3 = 18 ✓

## Common Traps

- **(a + b)² ≠ a² + b²**: (a+b)² = a² + 2ab + b²
- **√(a² + b²) ≠ a + b**: √(9 + 16) = √25 = 5, not 3+4=7
- **Negative base with even exponent**: (−3)² = 9 (positive), but −3² = −9 (the negative is outside)
- **Fractional exponents**: 8^(2/3) ≠ 8² ÷ 3. Do root first: ∛8 = 2, then square: 4.
